# Terminal Dungeon

A terminal-based dungeon game written in C++. 

## Features

- Battle System
- Inventory/Equipment System
- Economy/Shop System 
- 10 stages/levels
- RNG
- Terminal Interface

## Requirements

- Linux
- g++
- Make

## Build

```bash
make
```

## Run

```bash
make run
```

## Notes

For manual build and run, instructions are in `Makefile`.
