# Terminal Dungeon

A terminal-based dungeon game written in C++. 

## Features

- Battle System (WIP)
- Inventory System (WIP)
- Economy System (WIP)
- 10 stages/levels (WIP)
- RNG (WIP)
- Terminal Interface

## Requirements

- Linux
- g++
- Make

## Build

```bash
make
```

## Run

```bash
make run
```

## Notes

For manual build and run, instructions are in `Makefile`.
